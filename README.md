# Simple Interest Calculator

A simple tool to calculate the interest earned or paid on a principal amount over a specific period at a given interest rate.

## Description
This project provides a straightforward way to calculate Simple Interest using the standard mathematical formula. It is designed to help users understand the basic concepts of financial interest.

## Formula
The calculation is based on the formula:
$$A = P(1 + rt)$$

Where:
* **P** = Principal amount (the initial sum of money)
* **r** = Annual interest rate (decimal)
* **t** = Time period (in years)

## Features
* Input for Principal, Rate, and Time.
* Instant calculation of the total interest and final amount.
* Clean and readable documentation.

## How to Use
1. Enter the initial amount you want to invest or borrow.
2. Enter the annual interest rate as a percentage.
3. Enter the number of years.
4. The calculator will output the total interest accrued over that period.
